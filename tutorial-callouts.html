Using Callouts

Learn how to use and respond to callouts in LSPDFR:

  1. Install callout plugins from trusted sources.
  2. Load into GTA V with LSPDFR enabled.
  3. Wait for callouts to appear or force them using the menu.
  4. Respond to callouts and follow on-screen instructions.
