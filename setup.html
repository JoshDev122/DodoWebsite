Setup Guide

Step-by-step instructions on how to install LSPDFR and its required tools.

  1. Install a clean copy of GTA V (Steam preferred).
  2. Download and install RagePluginHook.
  3. Download LSPDFR from the official site.
  4. Place files in your GTA V directory.
  5. Launch with RagePluginHook and complete the setup wizard.
